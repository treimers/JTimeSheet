# Getting Started

## First steps

1. **Manage** – Open **Manage** (menu or toolbar) and add at least one **Customer**. Under the customer, add **Projects** and **Tasks** as needed.
2. **Add activity** – Use **Add Activity** (menu or toolbar) or the hourly reminder to record your first activity. Choose customer, project, task, and set start and end time.
3. **Filter** – Use the filter bar to show only the customer, project, or date range you need. The table shows duration and daily totals.
4. **Export** – Use **Write Timesheet** to export the filtered activities to Excel (if the customer has an existing template configured in Manage), or **Export CSV** for a simple CSV file.

## Tips

- Use **Create new view** to save a filter combination (e.g. one view per customer) and switch quickly via tabs.
- Set **Reminder** and **Time grid** in **Settings** to get reminders and rounded time entry.
